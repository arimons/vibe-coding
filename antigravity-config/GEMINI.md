# Antigravity Global Rules

## 언어
- 모든 응답은 한국어로 작성
- 사고 과정, PRD, 중간 산출물, 에러 분석 모두 한국어
- 코드 주석도 한국어
- 변수명/함수명은 영어 유지 (코드 컨벤션)

## 응답 방식
- 작업 시작 전 계획을 먼저 간단히 설명하고 진행
- 에러 발생 시 원인과 해결책을 함께 제시
- 코드 변경 시 무엇을 왜 바꿨는지 설명
- 불확실한 부분은 추측하지 말고 명시적으로 질문

## 코딩 스타일
- Python: PEP 8 준수
- 함수/클래스마다 한국어 docstring 작성
- 매직 넘버 금지, 상수로 분리
- 파일 상단에 목적 주석 작성

## 안전 규칙
- 파일 삭제/덮어쓰기 전 반드시 확인 요청
- .env 파일 및 API 키 절대 코드에 하드코딩 금지
