# Week 3 — CLI 스크립트를 Streamlit GUI로

> **핵심**: Streamlit 코드를 직접 짜는 게 아니라, AI한테 시켜서 GUI를 만드는 경험입니다.
> 코드는 몰라도 됩니다. 결과가 작동하면 성공입니다.

---

## 왜 Streamlit인가?

CLI 스크립트는 강력하지만 불편합니다.

```
CLI 방식:
터미널 열기 → 폴더 이동 → python rename.py 입력 → 결과 확인
→ 동료한테 전달 불가, 매번 터미널 필요

Streamlit 방식:
브라우저에서 버튼 클릭 → 결과 확인
→ 누구나 사용 가능, 터미널 불필요
```

**같은 기능, 완전히 다른 경험입니다.**

---

## 사전 준비

```bash
# 가상환경 생성
python -m venv venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Mac/Linux)
source venv/bin/activate

# 패키지 설치
pip install streamlit
```

실행:
```bash
streamlit run app.py
```
→ 브라우저에서 `http://localhost:8501` 자동으로 열립니다.

## 실습 1: 파일 Renamer

### 무엇을 만드나?

실험 파일명이 제각각일 때, 규칙에 맞게 일괄 변경하는 앱입니다.

```
변경 전:                변경 후:
data.xlsx          →   20250301_data.xlsx
result final.csv   →   20250301_result_final.csv
IMG_0042.png       →   20250301_IMG_0042.png
```

### 화면 구성

```
┌─────────────────────────────────────┐
│  📁 파일 이름 정리기                 │
│                                     │
│  폴더 경로                          │
│  [C:\Users\연구원\실험데이터      ] │
│                                     │
│  앞에 붙일 텍스트                   │
│  [20250301_                       ] │
│                                     │
│  [미리보기]                         │
│  ─────────────────────────────────  │
│  data.xlsx        → 20250301_data.. │
│  result final.csv → 20250301_resu.. │
│  IMG_0042.png     → 20250301_IMG_.. │
│                                     │
│  [✅ 변경 실행]                      │
│                                     │
│  ✓ 3개 파일 이름 변경 완료          │
└─────────────────────────────────────┘
```

### AI 프롬프트 (Antigravity에 그대로 붙여넣기)

```
폴더 경로와 앞에 붙일 텍스트를 입력받아서
해당 폴더의 모든 파일 이름을 일괄 변경하는 Streamlit 앱을 만들어줘.

기능:
1. 폴더 경로 입력창
2. 앞에 붙일 텍스트 입력창 (예: 날짜_)
3. 미리보기 버튼: 변경 전 → 변경 후 표로 보여주기
4. 실행 버튼: 실제 파일명 변경
5. 완료 메시지

주의:
- 미리보기를 먼저 확인한 후에만 실행 가능하게 해줘
- 변경 전에 확인 체크박스 추가
```

## 실습 2: 파일 정리기

### 무엇을 만드나?

바탕화면이나 다운로드 폴더처럼 파일이 뒤섞인 곳을 확장자별로 자동 분류하는 앱입니다.

```
정리 전 (다운로드 폴더):        정리 후:
논문.pdf                    →   /PDF/논문.pdf
실험결과.xlsx               →   /Excel/실험결과.xlsx
발표자료.pptx               →   /PPT/발표자료.pptx
사진.png                    →   /Images/사진.png
```

### 화면 구성

```
┌─────────────────────────────────────┐
│  🗂️ 파일 자동 분류기                │
│                                     │
│  정리할 폴더                        │
│  [C:\Users\연구원\Downloads       ] │
│                                     │
│  [현황 분석]                        │
│  ─────────────────────────────────  │
│  📄 PDF    8개                      │
│  📊 Excel  3개                      │
│  🖼️ Images 12개                    │
│  📁 기타   5개                      │
│                                     │
│  [ ] 하위 폴더도 포함               │
│                                     │
│  [🗂️ 정리 시작]                    │
│                                     │
│  ✓ 28개 파일 분류 완료             │
└─────────────────────────────────────┘
```

### AI 프롬프트

```
지정한 폴더의 파일들을 확장자별로 자동 분류하는 Streamlit 앱을 만들어줘.

기능:
1. 폴더 경로 입력창
2. 현황 분석 버튼: 확장자별 파일 개수 표로 보여주기
3. 하위 폴더 포함 여부 체크박스
4. 정리 시작 버튼: 확장자별 폴더 생성 후 이동
   - PDF → /PDF
   - xlsx, csv → /Excel
   - png, jpg → /Images
   - pptx → /PPT
   - 나머지 → /기타
5. 완료 후 결과 요약 메시지
```

## 실습 3: 스크립트 실행기

### 무엇을 만드나?

터미널 없이 버튼 클릭만으로 Python 스크립트를 실행하는 앱입니다. 비기술자 동료도 쓸 수 있어요.

```
기존 방식:
터미널 → python crawl.py → 결과 확인
→ 동료는 못 씀

Streamlit 방식:
버튼 클릭 → 실행 → 결과 화면에 표시
→ 누구나 씀
```

### 화면 구성

```
┌─────────────────────────────────────┐
│  ⚙️ 자동화 스크립트 실행기          │
│                                     │
│  실행할 작업 선택:                  │
│  ○ 특허 데이터 수집                 │
│  ● 실험 데이터 정리                 │
│  ○ 보고서 생성                      │
│                                     │
│  파라미터:                          │
│  시작 날짜 [2025-01-01]             │
│  종료 날짜 [2025-03-01]             │
│                                     │
│  [▶ 실행]                          │
│  ─────────────────────────────────  │
│  실행 로그:                         │
│  ✓ 데이터 로드 완료 (143개 행)     │
│  ✓ 중복 제거 완료 (12개 제거)      │
│  ✓ 저장 완료: output_20250301.xlsx │
└─────────────────────────────────────┘
```

### AI 프롬프트

```
여러 Python 스크립트를 버튼 클릭으로 실행할 수 있는
Streamlit 앱을 만들어줘.

기능:
1. 실행할 스크립트 선택 (라디오 버튼)
2. 날짜 범위 입력창 (시작일, 종료일)
3. 실행 버튼
4. 실행 로그를 실시간으로 화면에 표시
5. 완료 후 결과 파일 다운로드 버튼

스크립트 목록:
- data_clean.py: 실험 데이터 정리
- report_gen.py: 보고서 생성
```
