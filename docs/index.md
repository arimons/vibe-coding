## 📋 강의 개요

### **목표**
반복 작업 자동화 도구를 직접 만들 수 있는 연구원

### **기간**
12주 (주 1회, 2시간)

### **구조**
- 1-6주: 기초 개념 + 실습
- 7-8주: 프로젝트 기획 + 심화 개념
- 9-12주: 개인 프로젝트 개발

### **핵심 철학**
- 코드 직접 작성 ❌
- AI에게 질문하는 법 ✅
- 개념 이해 + 도구 활용 ✅

---

## 📅 주차별 커리큘럼

### **Week 1: 기초 개념 + 환경 설정**

📖 참고 자료: [CLI 기초 가이드](cli-guide) · [용어 사전](glossary)

#### 주제
- 용어 사전 (프론트엔드/백엔드/API/프레임워크/기술스택)
- 프로그램 구조 이해 (CLI / GUI / Web Application)
- CLI 기초 명령어 (`cd`, `ls`, `python 파일명.py`)

#### 시연 (40분)
1. 터미널 열기 및 폴더 이동
2. Python 파일 실행
3. 가상환경 만들기 (`python -m venv venv`)
4. 패키지 설치 (`pip install pandas`)
5. 에러 발생 → 해결 과정 시연

#### 실습 (40분)
- Python 설치 확인 (`python --version`)
- VS Code 설치 및 터미널 열기
- hello.py 작성 및 실행
- pip로 패키지 설치

#### 과제
터미널에서 Python 파일 실행 3번 연습

---

### **Week 2: AI 질문법 + Python 최소 개념**

#### 주제
- 좋은 질문 vs 나쁜 질문
- 문제 쪼개기 (큰 문제 → 작은 단계들)
- Python 최소 개념 (변수, 함수, 에러 읽기)

#### 시연 (40분)
1. AI에게 "폴더의 모든 PDF 파일명을 list.txt에 저장" 요청
2. 받은 코드 저장 및 실행
3. 에러 발생 → 메시지 복사 → AI 재질문 → 수정

#### 실습 (40분)
- AI에게 .txt 파일명 출력 스크립트 요청
- 에러 나면 메시지 복사 → AI 재질문
- 성공할 때까지 반복

#### 과제
AI에게 간단한 파일 처리 스크립트 3개 요청

---

### **Week 3: 엑셀 처리 개념**

#### 주제
- pandas 가능성 이해 (읽기/쓰기/필터/정렬/병합/그래프)
- 내 엑셀 구조 파악하기
- 단계별로 쪼개서 요청하기

#### 시연 (40분)
여러 엑셀 파일 병합 - 3단계로 쪼개서 진행

#### 실습 (40분)
본인 엑셀 파일로 단계별 요청 → 실행 → 확인

#### 과제
본인 업무의 엑셀 반복 작업 1개 자동화

---

### **Week 4: 크롤링 - 개발자 도구**

📖 참고 자료: [프론트엔드 & 백엔드 기초개념](frontend-backend)

#### 주제
- HTML/CSS 최소 개념 (태그 구조, 클래스, id)
- 개발자 도구 F12 (Elements 탭, selector 찾기)
- 크롤링 도구 (requests, beautifulsoup, selenium)

#### 시연 (40분)
네이버 뉴스 F12 → selector 복사 → AI에게 크롤링 요청 → CSV 저장

#### 실습 (40분)
관심 사이트(특허청, 저널 등)에서 직접 selector 찾고 크롤러 만들기

#### 과제
정기적으로 확인하는 사이트 1개 크롤러 만들기

---

### **Week 5: CLI → Streamlit (GUI의 편함)**

#### 주제
- CLI의 불편함 vs Streamlit의 편함
- Streamlit 구조 (`st.file_uploader`, `st.button`, `st.write`)

#### 시연 (40분)
같은 크롤러를 CLI 버전 → Streamlit 버전으로 변환 비교

#### 실습 (40분)
Week 4 크롤러를 Streamlit으로 변환 (`streamlit run app.py`)

#### 과제
본인 스크립트를 Streamlit으로 변환

---

### **Week 6: Streamlit 심화**

#### 주제
- 파일 업로드 (`st.file_uploader`)
- 그래프 (`st.line_chart`, `st.bar_chart`, plotly)
- 결과 다운로드 버튼
- 레이아웃 (`st.columns`, `st.sidebar`)

#### 시연 (40분)
엑셀 처리 웹앱 단계별로 기능 추가

#### 실습 (40분)
본인 프로젝트에 파일 업로드 + 시각화 + 다운로드 추가

#### 과제
Streamlit 앱 완성도 높이기

### **Week 7: 프로젝트 기획**

📖 참고 자료: [Git 초보자 가이드](git-guide)

#### 시연 (30분)
예시: 논문 PDF 자동 정리 프로젝트 기획 전 과정 시연

#### 실습 (60분)

프로젝트 기획서 완성 (입력/처리/출력 명확히)
